# DBEvo

> Database schema migrations for Python. Inspired by Play Framework Evolutions.

## ✨ Особенности

- 📝 **Plain SQL** — пишите миграции на чистом SQL, без автогенерации
- ↕️ **!Ups / !Downs** — прямое и обратное применение в одном файле
- 🔒 **Безопасно** — транзакции, хеши, детект конфликтов
- 🚀 **Async-first** — построено на `asyncpg` для высокой производительности
- 🎯 **Простой CLI** — `dbevo apply`, `dbevo status`, `dbevo revert`

## 📦 Установка

```bash
pip install dbevo
```
